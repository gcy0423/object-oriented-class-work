# v10-teacher-student-ai-link

v10 第四阶段专门处理教师端联动。v9 完成后，学生端已经具备 AI-first 的完整闭环；v10 要让教师能看见、理解、干预和反馈学生端产生的学习过程证据。

## 版本目标

- 教师可查看学生 AI 行动执行情况。
- 教师可查看学生提交草稿、自检结果和最终提交差异。
- 教师可查看 AI 学习时间线和学习证据。
- 教师可基于学生 AI 数据生成干预建议。
- 教师可将干预发送给学生通知/提醒。
- 教师端学情分析接入学生 AI 过程数据。
- 教师端批改详情展示学生提交前 AI 自检结果。
- 教师端不直接替学生执行行动，只提供反馈和干预。

## 本版本不做

- 不重写教师端 `teacher-ai-first` 全量原型。
- 不做复杂多校区、多租户权限。
- 不允许教师修改学生 AI 历史记录。
- 不要求真实模型在线。
